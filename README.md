# 📈 주식 웹 페이지 📉

### 2022 광운대학교 데이터 베이스 및 시각화 3차 프로젝트 
### 주제 &nbsp; 👉 &nbsp; 금융 시장 플랫폼 제작

#### 팀원 &nbsp; 👦 &nbsp; 이현지 &nbsp; 박건우 &nbsp; 임강호 &nbsp; 한서진

<br />

## 📃 구현 기능

```
✅ 관리자 기능
 - 회원 관리, 주식회사 등록, 게시판 관리, 주식회사 정보 변경 등
✅ 이용자 기능
 - 회원가입 및 개인 정보 조회, 게시글 작성 등
 - 관심 종목, 보유 주식 조회 등
✅ 주식회사 정보 조회
✅ 주식 매매 기능
✅ 종목 토론 게시판
✅ 데이터 시각화
```
